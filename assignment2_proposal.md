Title: Customer Segmentation using Machine Learning

Problem Statement:
Businesses need to understand their customers better to improve marketing strategies. This project aims to group customers based on their purchasing behavior.

Dataset:
Mall Customer Segmentation Dataset (contains Age, Gender, Annual Income, Spending Score)

Algorithm Used:
K-Means Clustering (unsupervised learning)

Approach:

Load dataset
Select important features (Income, Spending Score)
Apply K-Means clustering
Identify customer groups

Expected Output:

Customers divided into clusters (e.g., high spenders, low spenders)
Visualization of clusters using graphs

Tools Used:
Python, Pandas, Matplotlib, Scikit-learn
